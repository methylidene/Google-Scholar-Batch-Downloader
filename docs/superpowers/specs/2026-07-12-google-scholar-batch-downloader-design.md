# Google Scholar 批量论文助手设计规格

## 目标与范围

开发 Chrome Manifest V3 扩展，在 Google Scholar 当前搜索结果页选择论文、按作者筛选、下载页面明确提供的开放 PDF，并默认导出 RIS 与 BibTeX。扩展同时提供发送到本机 Zotero 的入口；Zotero 不可用时不影响下载和导出。

项目最终位于 `E:\\B-CSTools\\0-Projects\\插件\\google-scholar-batch-downloader`。首版不自动翻页、不规避验证码、不绕过付费墙，也不使用可疑论文镜像。找不到开放 PDF 的论文仍导出元数据并标记为“仅元数据”。

## 用户界面

内容脚本在 Scholar 结果页顶部插入工具栏，并在每条结果旁添加复选框和状态标识。工具栏提供作者名筛选（不区分大小写、按包含关系匹配当前页作者字段）、全选、取消全选、仅选择有 PDF、已选数量、任务进度、“下载并导出”和“发送到 Zotero”。条目状态包括 `PDF`、`仅元数据`、`已选择`、`下载中`、`成功` 和 `失败`。

## 架构

### 页面采集器

将每条结果解析为统一结构：标题、作者列表、年份、来源、摘要片段、详情 URL、PDF URL、DOI（若页面可识别）和状态。解析器与界面代码分离，以便用固定 DOM 样本测试。

### 页面工具栏

管理当前页筛选、选择和状态展示，只操作解析后的论文模型，不承担下载或导出逻辑。

### 后台任务模块

Chrome Service Worker 使用 `chrome.downloads` 保存文件并维护逐条结果。下载请求适当错开；单篇失败不终止批次，最多重试一次。

### 导出与 Zotero 模块

在浏览器本地生成 RIS、BibTeX 和 JSON 结果清单。默认操作是下载 PDF 并同时生成 RIS 与 BibTeX。Zotero 操作使用独立按钮，优先与已安装的 Zotero Connector/本机 Zotero 协作；检测不到可用入口时提示用户使用已生成的 RIS 或 BibTeX，不把 Connector 视为硬依赖。

### 可选开放文献查询

核心功能只使用 Scholar 页面已有的开放 PDF。设置中预留默认关闭的 DOI/开放文献 API 补充查询接口；接口失败或未配置时不得影响核心功能。

## 数据流

1. 页面加载后解析当前结果页。
2. 用户按作者筛选或手动勾选论文。
3. 点击“下载并导出”后将规范化条目发送给后台。
4. 后台为可用 PDF 创建下载任务，同时生成批次 RIS、BibTeX 和 JSON 清单。
5. PDF 命名为 `第一作者 - 年份 - 标题.pdf`；Windows 非法字符替换为空格，连续空白折叠，并限制到安全长度。
6. 同名文件使用 Chrome 唯一化命名，不覆盖已有文件。
7. 无 PDF 条目保留在元数据文件中并标记为“仅元数据”。
8. “发送到 Zotero”独立执行，其成败不改变下载与导出结果。

## 错误处理

- 单篇下载失败最多重试一次，并记录最终原因。
- 页面结构不符合预期时停止批量操作并提示可能已变化。
- 检测到验证码或异常流量页面时不继续请求。
- 缺失作者、年份或 DOI 不阻止导出；缺失字段使用空值，不伪造数据。
- Zotero 不可用时提示 RIS/BibTeX 备用方案。
- 可选 API 超时、限流或无结果时回退为页面数据。

## 权限与隐私

申请 `downloads`、`storage`、`activeTab` 和 Google Scholar 必要站点权限。只有启用开放文献补充查询时才使用相应 API 域名权限。扩展不采集或上传搜索记录，不读取 Scholar 之外的普通页面，不保存 Google 或 Zotero 登录凭据。解析、文件名处理和文献文件生成均在浏览器本地完成。

## 测试策略

- 单元测试：作者解析与匹配、年份提取、文件名清理与长度限制、RIS/BibTeX 转换、缺失字段和重复命名策略。
- DOM 固定样本测试：中英文结果、有无 PDF、多个作者、缺失年份、异常条目和页面结构不匹配。
- Chrome 手工验收：当前页全选/取消、作者筛选、仅 PDF 筛选、批量下载、失败重试、RIS/BibTeX 导入 Zotero，以及 Zotero 可用和不可用两种情况。

## 完成标准

- 当前页论文可稳定识别、勾选和按作者筛选。
- 页面明确提供的开放 PDF 可批量下载，单篇失败不终止批次。
- 默认生成有效的 RIS、BibTeX 和 JSON 结果清单。
- PDF 命名规则兼容 Windows。
- Zotero 可用时提供一键发送；不可用时清楚提示备用流程。
- 不自动翻页、不规避验证码、不绕过付费墙。

## 非目标

自动遍历多页结果；绕过订阅、机构登录、付费墙或验证码；从未经授权的镜像站下载；首版提供云同步、用户账户或远程任务服务。
